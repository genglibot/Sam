Selected Publications

UniRiT: Towards Few-Shot Non-Rigid Point Cloud Registration.
Geng Li, Haozhi Cao, Mingyang Liu, Chenxi Jiang, Jianfei Yang.

Under review [arxiv] [paper]
The MedMatch3D dataset of real organ point clouds is introduced, addressing the few-shot problem by decomposing non-rigid motion and proposing a novel method for few-shot N-PCR.

GERA: Geometric Embedding for Efficient Point Registration Analysis.
Geng Li, Haozhi Cao, Mingyang Liu, Shenghai Yuan, Jianfei Yang.

International Conference on Robotics and Automation(ICRA 2025) [arxiv] [paper] [video]
GERA introduces an offline geometric encoder for efficient non-rigid point cloud registration, using supervised training to minimize differences in high-dimensional geometric features instead of absolute positions.

LCNet: A Robust and Accurate Non-Rigid 3D Point Set Registration Approach for Image-Guided Liver Surgery.
Mingyang Liu, Zhe Min, Geng Li, Hao Yu, Rui Song, Yibin Li, Max Q.-H. Meng.

Under review [paper ]
This work presents LCNet, an unsupervised learning-based method for non-rigid 3D point set registration, specifically designed for enhancing image-guided liver surgery.
